# 🛡️ 图像水印隐藏系统

> 基于 QIM 量化调制的盲水印系统 — 图中藏图 · 鲁棒水印 · 攻击测试

## ✨ 功能

- 🔒 **图像隐藏** — 将水印/秘密图嵌入封面图，肉眼不可见
- 👁️ **盲提取** — 提取水印不需要原始图像
- ⚔️ **9种攻击模拟** — 噪声、压缩、旋转、缩放、裁剪等
- 📊 **实时指标** — PSNR / SSIM / NCC 一键评估

## 🧠 算法

| 方法 | PSNR | 特点 |
|------|------|------|
| **QIM-LL 小波盲水印** | ~31 dB | DWT 低频子带 QIM，抗攻击最强 |
| **色度通道水印** | ~36 dB | Cb 通道嵌入，完全不可见 |
| **分块 DCT 水印** | ~39 dB | JPEG-like 8×8 DCT，极隐蔽 |
| **SVD 奇异值水印** | ~30 dB | 抗几何变换 |

### 为什么选 QIM

- **量化调制**：将系数强制归入奇/偶区间，天然抗噪声
- **盲提取**：判断奇偶即可，不需要原图对比
- **LL 子带嵌入**：低频系数大（数百量级），量化稳定

```
系数 c → floor(c/δ) → 奇数区间 = 比特1，偶数区间 = 比特0
攻击后 c+ε → floor((c+ε)/δ) → ε < δ/2 则奇偶不变 ✓
```

## 📡 API

### `POST /api/hide`
嵌入水印

| 参数 | 类型 | 说明 |
|------|------|------|
| `cover` | file | 封面图像 |
| `secret` | file | 水印图像 |
| `method` | string | `qim` / `chroma` / `dct` / `svd` |
| `param` | float | 嵌入强度 |

返回：容器图、提取结果、差异图、PSNR/SSIM/NCC

### `POST /api/full-pipeline`
完整流水线：隐藏 → 攻击 → 提取 → 评估

### `POST /api/attack`
对图像施加攻击（支持多选）

### `POST /api/evaluate`
对比两张图的 PSNR / SSIM

## 🎛️ 参数指南

### QIM-LL 量化步长 δ

| δ | PSNR | NCC(JPEG50) | 适用场景 |
|----|------|-------------|---------|
| 8~12 | >35 dB | <0.3 | 纯隐蔽传输 |
| **16~24** | **29~32 dB** | **0.7~0.9** | **均衡之选** |
| 28~36 | 25~28 dB | >0.95 | 强鲁棒优先 |
| 40+ | <25 dB | ≈1.0 | 可见水印 |
